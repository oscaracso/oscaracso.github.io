App Write Up


In this project andrew and I created an app called Pokedodge. In the creation of this app we used block coding on thunkable.
The app includes components that allow it to call and play sound when a specific event or button is pressed. The point of the game is to collect pokeballs to gain points and the gold one subtracts points from your total.
We set a timer to stop the game and switch screens once it goes off. The screen it switches to is the score screen, and there is also a song that is played once the screen initializes. It pulls the data of the player score from the tiny database. The game screen saves the player score and top score when the timer goes off to switch screens. Overall on this project i say that Andrew and I did 50/50 on everything. We both put some time and thought into the creation of this app.
